# Programming Language Configurations

Configures programming language toolchains and development environments.

## Supported Languages

- C# (csharp.nix)
- Elixir
- Java
- JavaScript/Node.js
- Lua
- Nix
- Python
- Ruby
- Rust

## Purpose

Each file sets up the complete development environment for its respective language, including compilers, interpreters, package managers, LSP servers, and common tools.
