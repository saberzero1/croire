# Home Manager NixOS Configuration

User-level configuration specific to NixOS systems.

## Structure

- `programs/` - NixOS-specific program configurations
- `services/` - User services for NixOS
- `settings/` - NixOS user preferences and environment settings

## Purpose

Contains user-level configurations that only apply when running on NixOS, such as Wayland programs, Hyprland setup, and Linux-specific tools.
