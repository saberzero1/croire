# Shared Home Directory Settings

Cross-platform home directory configuration.

## Features

- `activation.nix` - Home Manager activation scripts
- `files/` - File and directory management
- `session-path.nix` - PATH configuration for executables
- `session-variables.nix` - Environment variables (editor, shell, tool settings)
- `shell-aliases.nix` - Comprehensive shell aliases for common tasks

## Purpose

Provides consistent home directory setup across platforms with shared aliases for Git, file navigation, development tools, and utilities.
