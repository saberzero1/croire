# LazyVim Configuration

Comprehensive Neovim configuration using LazyVim framework.

## Structure

- `languages/` - Language-specific plugins and configurations
- `plugins/` - Custom plugin definitions
- Core configurations for AI, coding, DAP debugging, editor settings, formatting, keymaps, linting, testing, and UI

## Purpose

Provides a full-featured Neovim setup with LSP support, completion, debugging, and extensive plugin ecosystem managed by LazyVim's lazy-loading plugin manager.
