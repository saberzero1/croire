# Shared Home Manager Configuration

User-level configuration shared across all platforms (NixOS and macOS).

## Structure

- `languages/` - Programming language configurations and toolchains
- `programs/` - Cross-platform program configurations (editors, shells, utilities)
- `services/` - User services that work on all platforms
- `settings/` - Shared user preferences and environment settings

## Purpose

Contains configurations that apply to all systems regardless of platform, ensuring consistent user experience and tooling across NixOS and macOS environments.
