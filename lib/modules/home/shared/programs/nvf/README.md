# Nvf (Neovim-Flake) Configuration

Comprehensive Neovim configuration using the nvf (neovim-flake) framework.

## Structure

- `languages/` - Language-specific configurations
- Core modules for assistant, autocomplete, autopairs, coding, comments, dashboard, debugger, diagnostics, filetree, formatter, git, keymaps, LSP, navigation, snippets, statusline, tabline, terminal, theme, treesitter, and UI

## Purpose

Provides a declarative, reproducible Neovim configuration using Nix, with extensive support for LSP, debugging, and modern editor features.
