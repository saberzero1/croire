# Darwin System Configuration

Contains system-level configuration for macOS.

## Structure

- `fonts.nix` - System font configuration
- `packages.nix` - System-wide package installations
- `security.nix` - Security and privacy settings
- `settings.nix` - macOS system preferences and defaults

## Purpose

These modules configure system-wide settings that affect the entire macOS system, including Finder, Dock defaults, accessibility, security, and global preferences.
